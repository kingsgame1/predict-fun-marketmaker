## ⚙️ 配置选项

### 积分优化配置

```bash
# 核心积分优化
export MM_POINTS_PRIORITIZE=true              # 优先积分市场（默认）
export MM_POINTS_OPTIMIZATION=true             # 启用积分优化（默认）
export MM_POINTS_V2_OPTIMIZER=true             # V2 极致优化（默认）
export MM_ONLY_POINTS_MARKETS=true             # 只做积分市场

# 积分规则
export MM_POINTS_MIN_SHARES=100                # 最小订单股数
export MM_POINTS_MAX_SPREAD_CENTS=6            # 最大价差（美分）
export MM_POINTS_ASSUME_ACTIVE=true            # 启用默认规则

# Probable 积分适配器
export PROBABLE_POINTS_ENABLED=true             # 启用 Probable 积分（默认）
export PROBABLE_MIN_SHARES=50                   # Probable 最小订单
export PROBABLE_MAX_SPREAD_CENTS=5              # Probable 最大价差
export PROBABLE_QUALITY_THRESHOLD=0.7           # 质量阈值
```

### V2 优化器配置

```typescript
// 默认配置（已在代码中设置）
{
  weights: {
    points: 0.4,      // 积分权重 40%
    profit: 0.3,      // 利润权重 30%
    risk: 0.2,        // 风险权重 20%
    efficiency: 0.1,  // 效率权重 10%
  },
  mlParams: {
    sizeImpactFactor: 0.5,      // 订单大小影响
    spreadImpactFactor: 0.3,    // 价差影响
    liquidityInteraction: 0.2,  // 流动性交互
    volatilityPenalty: 0.1,     // 波动惩罚
    competitionPenalty: 0.15,   // 竞争惩罚
  }
}
```
