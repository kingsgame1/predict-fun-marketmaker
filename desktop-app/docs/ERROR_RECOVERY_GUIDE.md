# 错误恢复指南

## 常见错误及解决方案

### 🔴 配置错误

#### 错误：缺少必填配置项
```
❌ 缺少 API_KEY
❌ 缺少 PRIVATE_KEY
```

**解决方案**：
1. 点击 **"💡 智能建议"** 按钮获取详细建议
2. 或点击 **"✨ 一键最佳实践"** 自动配置
3. 填写必填项后保存

#### 错误：JWT 无效
```
❌ JWT Token 无效或已过期
```

**解决方案**：
1. 重新获取 JWT Token
2. 在配置中心更新 `JWT_TOKEN`
3. 保存配置并重启

---

### 🟡 运行时错误

#### 错误：订单提交失败
```
❌ 订单提交失败: Insufficient balance
```

**解决方案**：
1. 检查账户余额
2. 调整 `ORDER_SIZE` 参数
3. 或切换到模拟模式测试

#### 错误：WebSocket 连接断开
```
⚠️ WebSocket 连接已断开，正在重连...
```

**解决方案**：
1. 系统会自动重连
2. 检查网络连接
3. 如果持续断开，尝试重启应用

---

### 🟢 积分优化问题

#### 问题：积分效率低
```
积分效率: 45% (目标 80%+)
```

**解决方案**：
1. 点击 **"✨ 一键最佳实践"** 重新优化
2. 确保启用了以下选项：
   - `MM_POINTS_PRIORITIZE=true`
   - `MM_POINTS_V2_OPTIMIZER=true`
   - `ORDER_SIZE ≥ 100`
   - `MAX_SPREAD ≤ 0.06`

#### 问题：不符合积分规则
```
❌ 订单不符合积分规则
```

**解决方案**：
1. 使用 **"积分优化模板"**
2. 检查订单大小是否 ≥ 100
3. 检查价差是否 ≤ 6¢

## 🛠️ 一键恢复功能

### 恢复到最佳实践配置
点击 **"✨ 一键最佳实践"** 按钮，系统会：
- ✅ 自动配置所有积分优化参数
- ✅ 启用 WebSocket 实时数据
- ✅ 设置最优订单参数
- ✅ 启用 V2 优化引擎

### 恢复到最小配置
点击 **"加载最小模板"** 按钮，系统会：
- ✅ 只配置必填项
- ✅ 使用安全的默认值
- ✅ 适合新手用户

## 🔄 自动恢复功能

系统具有以下自动恢复功能：

### 1. 自动重连
- WebSocket 断线自动重连
- API 连接失败自动重试
- 最多重试 3 次

### 2. 自动降级
- 检测到高风险时自动降级
- 使用保守参数继续运行
- 记录降级原因

### 3. 自动暂停
- 余额不足时自动暂停
- 市场异常时自动暂停
- 可配置恢复条件

## 📋 常见问题 FAQ

### Q: 如何切换到简化版？
**A**: 点击右上角 **"🎯 简化版"** 按钮

### Q: 如何切换到完整版？
**A**: 点击右上角 **"🔧 完整版"** 按钮

### Q: 模板按钮不工作？
**A**:
1. 按 F12 查看控制台错误
2. 刷新页面重试
3. 或使用完整版的对应功能

### Q: 积分为 0？
**A**:
1. 检查是否启用了积分优化
2. 确认订单符合积分规则
3. 查看积分状态卡片了解详情
