## 📊 积分优化详解

### 什么是积分？

Predict.fun 平台会根据做市表现给予积分奖励。

### 积分规则

为了获得积分，订单需要满足：

1. **最小股数**：`ORDER_SIZE ≥ 100`
2. **最大价差**：`SPREAD ≤ 0.06`（6 美分）
3. **活跃市场**：在有积分规则的市场做市

### 积分效率

**积分效率** = 符合积分规则的订单数 / 总订单数

- **80%+**：优秀 ✅
- **60-80%**：良好 🟡
- **< 60%**：需要优化 🔴

### 如何提高积分效率？

1. **使用一键最佳实践**
2. **启用 V2 优化引擎**
3. **使用积分优化模板**
4. **启用 WebSocket 实时数据**

## 🎯 使用技巧

### 技巧 1：先用模拟模式测试
1. 使用 **"模拟模式"** 测试配置
2. 观察积分效率和做市指标
3. 确认无误后切换到 **"实盘模式"**

### 技巧 2：定期查看积分状态
1. 定期查看 **"积分状态"** 卡片
2. 确保积分效率保持在 **80%+**
3. 如果效率下降，重新点击 **"✨ 一键最佳实践"**

### 技巧 3：利用模板快速配置
1. 点击 **"积分优化模板"** 加载优化配置
2. 根据需要微调参数
3. 保存配置并启动

### 技巧 4：关注系统体检
1. 定期点击 **"系统体检"** 的 **"刷新"** 按钮
2. 确保所有检查项都正常
3. 如有问题，按提示修复
