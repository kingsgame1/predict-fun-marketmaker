# Predict.fun 做市商 & 套利机器人 - 快速使用指南

## 🚀 快速开始（3步配置）

### 第1步：获取 API Key
1. 加入 Discord: https://discord.gg/predictdotfun
2. 创建工单申请 API Key
3. 将 API Key 填写到配置中

### 第2步：配置钱包（最小配置）
```
API_KEY=你的API密钥
PRIVATE_KEY=你的钱包私钥
ENABLE_TRADING=false          # 模拟模式
ORDER_SIZE=10                 # 订单大小
```

### 第3步：启动测试
1. 点击"加载最小模板"按钮
2. 填写 API_KEY 和 PRIVATE_KEY
3. 点击"保存配置"
4. 点击"启动做市商"
5. 观察日志，确认订单正常创建

## 📊 功能说明

### 做市商（Market Maker）
- 自动提供买卖报价
- 赚取买卖价差（Spread）
- 管理库存风险

**启动方式**：点击"启动做市商"按钮

**关键参数**：
- `ORDER_SIZE`: 订单金额（USDT）
- `SPREAD`: 价差百分比（0.02 = 2%）
- `MAX_POSITION`: 最大持仓

### 套利机器人（Arbitrage）
- 监控不同市场价差
- 自动执行套利交易
- 支持跨平台套利

**启动方式**：点击"启动套利"按钮

**关键参数**：
- `ARB_AUTO_EXECUTE`: 是否自动执行
- `ARB_WS_REALTIME`: 实时扫描

## 🔴 实盘交易前必读

### 检查清单
- [ ] 已在模拟模式测试通过
- [ ] API Key 配置正确
- [ ] JWT Token 已填写
- [ ] 钱包有足够余额
- [ ] 了解风险控制参数
- [ ] 设置了最大亏损限制

### 推荐设置
```bash
ENABLE_TRADING=true           # 实盘模式
AUTO_CONFIRM=true            # 自动确认（谨慎）
ORDER_SIZE=5                 # 从小金额开始
MAX_POSITION=50             # 限制最大持仓
MAX_DAILY_LOSS=100          # 设置每日最大亏损
```

## 🛡️ 安全建议

1. **不要使用主钱包**
   - 建议创建专门的交易账户
   - 只存入必要的资金

2. **从小额开始**
   - 初始测试：ORDER_SIZE=5
   - 确认正常后再增加

3. **设置止损**
   - MAX_DAILY_LOSS：每日最大亏损
   - MAX_POSITION：最大持仓限制

4. **监控日志**
   - 定期检查交易日志
   - 留意异常交易
   - 监控账户余额

## 🐛 常见问题

### Q: 按钮点击没有反应？
A:
1. 检查配置是否已保存
2. 查看日志是否有错误
3. 确认 API Key 已填写

### Q: 做市商无法启动？
A:
1. 检查 API_KEY 和 PRIVATE_KEY
2. 确认网络连接正常
3. 查看日志错误信息

### Q: 没有看到订单？
A:
1. 确认不是模拟模式
2. 检查 ENABLE_TRADING 设置
3. 查看日志中的错误

### Q: 如何切换到实盘？
A:
1. 填写 JWT_TOKEN
2. 设置 ENABLE_TRADING=true
3. 设置 AUTO_CONFIRM=true
4. 保存配置并重启
