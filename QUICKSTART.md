# 🚀 快速开始指南

## 📖 一句话总结

Predict.fun 做市商现在有两个版本：
- **简化版（🎯）**：适合小白用户，一键优化，专注积分
- **完整版（🔧）**：适合高级用户，包含套利等所有功能

两个版本可以随时切换，无需重启应用！

---

## 🎯 首次使用

### 第1步：启动应用

```bash
npm start
```

### 第2步：选择版本

首次启动会看到版本选择页面：
- 点击 **"🎯 简化版"** - 推荐新手
- 点击 **"🔧 完整版"** - 高级用户

勾选 **"记住我的选择"**，下次自动进入。

### 第3步：优化配置（简化版）

如果选择了简化版：
1. 点击 **"✨ 一键最佳实践"** 按钮
2. 点击 **"保存配置"** 按钮

### 第4步：启动做市商

点击 **"启动做市商"** 按钮

### 第5步：监控积分

查看 **"积分状态"** 卡片，确认：
- ✅ 订单符合积分规则
- ✅ 积分效率高（80%+）

---

## 🔄 版本切换

### 任何时候都可以切换！

**简化版 → 完整版**：点击右上角 **"🔧 完整版"** 按钮
**完整版 → 简化版**：点击右上角 **"🎯 简化版"** 按钮

✅ 无需重启应用
✅ 做市商继续运行
✅ 配置和状态保持不变
