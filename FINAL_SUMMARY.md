## 🚀 使用流程

### 新手用户（简化版）

```bash
# 第1步：启动应用
npm start

# 第2步：选择版本
点击 "🎯 简化版"

# 第3步：一键优化
点击 "✨ 一键最佳实践"
点击 "保存配置"

# 第4步：启动做市商
点击 "启动做市商"

# 第5步：监控积分
查看 "积分状态" 卡片
```

### 高级用户（完整版）

```bash
# 第1步：启动完整版
npm start
选择 "🔧 完整版"

# 第2步：配置套利
在 "策略开关" 中启用：
- CROSS_PLATFORM_ENABLED=true
- ARB_PREFLIGHT_ENABLED=true
- ARB_WS_REALTIME=true

# 第3步：选择平台
在 "套利机会" 卡片中：
- 类型：选择 "跨平台套利"
- 平台：勾选要使用的平台
  ✅ Predict.fun
  ✅ Polymarket
  ✅ Opinion
  ✅ Probable

# 第4步：启动系统
点击 "启动做市商"
点击 "启动套利"

# 第5步：监控指标
查看 "执行指标" 卡片
查看 "套利机会" 卡片
```
