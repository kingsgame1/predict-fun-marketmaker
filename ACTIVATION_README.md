# 🔑 激活码系统 - 快速开始

## 📋 系统概述

**模块分离**:
- ✅ **做市商模块** - 完全免费，无需激活
- 🔒 **套利模块** - 需要激活码

## 👤 用户端使用

### 激活套利模块

```bash
# 方法1: 直接激活
npm run activate <激活码>

# 方法2: 交互式激活
npm run activate

# 方法3: 检查激活状态
npm run activate:check
```

### 桌面应用激活

1. 打开桌面应用（简化版）
2. 在"🤖 全自动套利"区块找到激活输入框
3. 输入激活码
4. 点击"激活"按钮

### 获取激活码

请联系管理员获取激活码。

## 🔒 安全说明

**重要**:
- 激活码绑定到机器硬件
- 每台电脑需要独立的激活码
- 激活码有固定的有效期
- 请勿分享激活码

详细安全说明: [docs/ACTIVATION_SECURITY.md](docs/ACTIVATION_SECURITY.md)

## ❓ 常见问题

**Q: 激活码无效?**
→ 检查激活码格式（XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX）
→ 联系管理员重新生成

**Q: 换电脑后无法使用?**
→ 激活码绑定到机器，需要为新电脑重新激活

**Q: 如何续费?**
→ 联系管理员获取新的激活码
