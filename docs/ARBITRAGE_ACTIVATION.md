# 🔑 套利模块激活说明

## 📋 功能说明

此项目包含两个模块：

- ✅ **做市商模块** - 完全免费，无需激活
- 🔒 **套利模块** - 需要激活码才能使用

## 🚀 如何激活

### 第1步：获取激活码

联系管理员获取激活码，提供以下信息：
- 用户ID（唯一标识符）
- 用户名（显示用）
- 需要的有效期（天数）

### 第2步：激活套利模块

**方法1 - 命令行激活**：

```bash
npm run activate <激活码>
```

按提示输入用户ID和用户名。

**方法2 - 交互式激活**：

```bash
npm run activate
```

选择"1. 激活许可证"，然后按提示操作。

**方法3 - 桌面应用**：

1. 打开桌面应用（简化版）
2. 在"🤖 全自动套利"区块找到激活输入框
3. 输入激活码
4. 点击"激活"按钮

### 第3步：验证激活

```bash
npm run activate:check
```

输出示例：
```
当前激活状态:
状态: ✅ 已激活
消息: 激活码有效

激活信息:
激活码: A7E5-2F9C-725E-1899-...
用户: John Doe
到期日期: 2027/2/23
剩余天数: 364 天
```

## ❌ 常见问题

### Q: 激活码无效？

**可能原因**：
- 激活码格式错误（应为 XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX）
- 激活码与机器不匹配
- 激活码已过期

**解决方法**：
- 检查激活码是否完整（包含7个连字符）
- 确认激活码未过期
- 联系管理员重新生成

### Q: 激活码在另一台电脑无法使用？

**原因**：激活码绑定到当前机器的硬件，每台电脑需要单独的激活码。

**解决方法**：联系管理员为每台机器生成独立的激活码。

### Q: 如何延长有效期？

联系管理员，提供当前激活信息，说明需要延长的天数。

### Q: 重装系统后激活失效？

**原因**：机器ID可能改变。

**解决方法**：
1. 保存 `.arbitrage_activation.json` 文件
2. 重装后恢复此文件
3. 或联系管理员重新激活
