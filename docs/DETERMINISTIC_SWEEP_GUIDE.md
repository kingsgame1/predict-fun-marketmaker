# 🎯 确定性尾盘套利策略 - 完整指南

## 策略概述

**核心思想**：对于有确定性结果的多结果市场，利用0手续费机制进行循环套利和对冲套利。

### 适用场景

1. **票房冠军预测**：多部电影竞争，只有一个冠军
2. **选举预测**：多个候选人，只有一个获胜者
3. **排名预测**：多个选项，只有一个第一
4. **任何有确定性结果的多结果市场**

### 核心优势

- ✅ **零手续费**：Probable市场0手续费，可以高频交易
- ✅ **确定性收益**：基于验证数据，接近100%胜率
- ✅ **循环放大**：通过循环交易做大交易量
- ✅ **双重收益**：循环套利 + 对冲套利

## 策略1：循环交易（获利99.9%）

### 工作原理

```
买一价买入 → 挂99.9卖出 → 重复 → 放大收益
```

### 执行步骤

**循环10次，每次100股：**

1. **第1次循环**：
   - 买入：100股 @ 0.95 = $95
   - 卖出：100股 @ 0.999 = $99.9
   - 利润：$4.9

2. **第2次循环**：
   - 买入：100股 @ 0.95 = $95
   - 卖出：100股 @ 0.999 = $99.9
   - 利润：$4.9

... 重复10次 ...

**总收益**：$4.9 × 10 = **$49**

### 为什么能盈利？

1. **零手续费**：Probable不收取交易费用
2. **买卖价差**：0.999 - 0.95 = 4.9% 利润空间
3. **循环放大**：虽然单次利润不大，但循环10次累积可观

## 常见问题

### Q1: 如果验证数据错了怎么办？

**A**: 极低概率（<1%），但可以通过以下方式降低风险：
1. 使用多个数据源交叉验证
2. 只在确定性>95%时执行
3. 设置止损（虽然概率很低）

### Q2: 99.9能成交吗？

**A**: 在接近结算时，99.9是合理价格：
- 买方愿意接受接近1的价格

- 流动性通常会很好

### Q3: 需要多少本金？

### Q4: 可以同时做多个市场吗？

**A**: 可以，但要注意：
1. 确保每个市场都有确定性数据
2. 不要超过总仓位限制
3. 建议每次只做1-2个市场
