# 界面内版本切换功能

## ✨ 新功能

现在可以在应用界面内直接切换完整版和简化版，无需重启应用！

---

## 🎯 如何切换

### 从完整版切换到简化版

1. **找到切换按钮**
   - 位置：界面右上角，状态指示器旁边
   - 按钮：🎯 简化版

2. **点击按钮**
   - 会弹出确认对话框
   - 显示做市商运行状态
   - 确认后立即切换

3. **切换效果**
   - ✅ 做市商继续运行（不受影响）
   - ✅ 所有配置保持不变
   - ✅ 界面立即切换到简化版
   - ✅ 日志记录切换操作

### 从简化版切换到完整版

1. **找到切换按钮**
   - 位置：界面右上角，状态指示器旁边
   - 按钮：🔧 完整版

2. **点击按钮**
   - 会弹出确认对话框
   - 显示做市商运行状态
   - 确认后立即切换

3. **切换效果**
   - ✅ 做市商继续运行（不受影响）
   - ✅ 所有配置保持不变
   - ✅ 界面立即切换到完整版
   - ✅ 可以使用套利功能

## 🔍 重要说明

### ✅ 保持不变的内容
- **做市商运行状态** - 切换界面不影响做市商
- **环境变量配置** - `.env` 文件共享
- **日志记录** - 所有日志保持一致
- **WebSocket连接** - 实时数据连接不中断

### ⚠️ 注意事项
1. **套利机器人**
   - 在简化版中看不到套利控制
   - 但如果正在运行，会继续运行
   - 切换到完整版后可以看到

2. **界面刷新**
   - 切换是立即生效的
   - 相当于重新加载页面
   - 未保存的修改会丢失

## 🔧 技术实现

### 切换原理
```javascript
// 点击切换按钮
window.location.href = 'index_simple.html';  // 切换到简化版
window.location.href = 'index.html';         // 切换到完整版
```

### 状态保持
```javascript
// 检查做市商状态
const status = window.electron.ipcRenderer.sendSync('status');
// status.marketMaker - 做市商是否运行
// status.arbitrage - 套利是否运行
```

### 日志记录
```javascript
// 记录切换操作
pushLog({
  type: 'system',
  level: 'system',
  message: '🎯 正在切换到简化版...'
});
```
