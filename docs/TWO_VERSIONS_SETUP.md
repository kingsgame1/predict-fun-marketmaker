## 🔄 版本切换

### 方式1：界面内切换（推荐）✨

直接在应用内点击右上角按钮切换：
- **完整版**：点击 🎯 简化版 按钮
- **简化版**：点击 🔧 完整版 按钮

**优势**：
- ✅ 无需重启应用
- ✅ 做市商继续运行
- ✅ 配置保持不变
- ✅ 自动显示运行状态

**详细文档**：`docs/VERSION_SWITCH_GUIDE.md`

### 方式2：命令行切换

两个版本共享同一个配置文件（`.env`），可以随时切换：

```bash
# 当前运行简化版，想切换到完整版
# 1. 停止当前应用（Cmd+Q 或关闭窗口）
# 2. 启动完整版
npm start

# 当前运行完整版，想切换到简化版
# 1. 停止当前应用
# 2. 启动简化版
npm run start:simple
```
