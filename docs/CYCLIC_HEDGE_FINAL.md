## 🔄 完整循环流程

### 阶段 1: 初始建仓
```
状态：0 YES + 0 NO（空仓）
挂单：YES 卖单 + NO 卖单（第二档）
目标：赚取积分 ✨

⚡ 风险事件：YES 卖单被吃
  → 被迫买入 10 YES
  → 持仓：10 YES + 0 NO ❌

🛡️ 立即对冲：
  → 市价买入 10 NO
  → 持仓：10 YES + 10 NO ✅
  → 状态：1:1 完美对冲
```

### 阶段 2: 持续做市
```
状态：10 YES + 10 NO（对冲状态）
挂单：YES 卖单 + NO 卖单（第二档）
目标：继续赚取积分 ✨

⚡ 风险事件：NO 卖单被成交
  → 卖出 10 NO
  → 持仓：10 YES + 0 NO ❌

🔄 立即平仓：
  → 市价卖出 10 YES
  → 持仓：0 YES + 0 NO ✅
  → 状态：回到空仓
```

### 阶段 3: 重新开始
```
回到阶段 1，重复循环 ♻️

每个周期独立，风险不会累积！
```

## ✅ 核心优势

### 1. 风险隔离

每个周期独立：
```
周期1: 0 → 对冲 → 平仓 → 0
周期2: 0 → 对冲 → 平仓 → 0
周期3: 0 → 对冲 → 平仓 → 0
...

风险不会累积到下一周期！
```

### 2. 积分为主

```
主要收益：积分（挂单奖励）✨
次要收益：价差（被吃单后）

即使亏钱，积分收益也能覆盖！
```

### 3. 完全对冲

```
1:1 持仓时：
  YES + NO = 1

价格变化：
  $0.60 + $0.40 = $1.00
  $0.80 + $0.20 = $1.00 ✅
  $0.30 + $0.70 = $1.00 ✅

价值不变，无方向性风险！
```

### 4. 自动循环

```
系统自动：
  ✅ 检测被吃单
  ✅ 立即对冲
  ✅ 继续挂单
  ✅ 检测成交
  ✅ 立即平仓
  ✅ 重新开始

无需人工干预！
```

## 📋 收益分析

### 单个周期收益

```
假设：
  - 订单大小：10 股
  - YES 价格：$0.60
  - NO 价格：$0.40
  - 挂单价差：1.5%

步骤1: 被吃单（第一次）
  成本：买入 10 YES @ $0.60 = $6
  对冲：买入 10 NO @ $0.40 = $4
  总成本：$10

步骤2: 继续做市
  积分收益：✨（主要收益）
  持仓价值：$10（不变）

步骤3: 被成交（第二次）
  卖出 10 NO @ $0.40 × 1.015 = $4.06
  卖出 10 YES @ $0.60 × 1.015 = $6.09
  总收入：$10.15

净利润：
  价差收益：$10.15 - $10 = $0.15
  积分收益：✨（无法量化）
  净收益：积分 + $0.15

结论：即使价差亏损，积分也能覆盖！
```

### 日收益估算

```
假设每天 5 个周期：

价差收益：$0.15 × 5 = $0.75
积分收益：✨✨✨（主要）

月收益估算：
  价差：$0.75 × 30 = $22.5
  积分：???（取决于平台奖励）

结论：主要靠积分，价差是 bonus！
```

## ⚠️ 风险警告

### 必须满足的条件

1. ✅ **YES + NO ≤ 1.05**
   - 实时监控，超标立即平仓

2. ✅ **高流动性**
   - 24小时交易量 > $10,000
   - 订单簿深度 > $1,000

3. ✅ **立即执行**
   - 被吃单后不能延迟
   - 市价单（不限制价）

### 常见错误

❌ **错误 1**: 在 YES+NO>1.05 的市场使用
- 后果：对冲失效

❌ **错误 2**: 对冲延迟
- 后果：暴露在方向性风险中

❌ **错误 3**: 忽略 YES+NO 变化
- 后果：可能从 1.00 涨到 1.20
