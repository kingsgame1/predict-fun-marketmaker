# 🔄 双版本系统说明

## ✅ 确认：系统有两个版本可以切换

### 版本对比

| 特性 | 完整版 | 简化版 |
|------|----------------------|------------------------|
| **文件** | `index.html` | `index_simple.html` |
| **大小** | ~40KB | ~7KB |
| **标题** | 做市与套利操作台 | 积分获取专用 |
| **按钮数量** | 50+个 | ~15个 |
| **开关数量** | 30个 | 3个 |
| **卡片数量** | 7个 | 5个 |
| **目标用户** | 高级用户 | 小白用户 |

## 🔄 切换方法

### 方法1：界面内切换（推荐）⭐⭐⭐⭐⭐

**位置**：界面右上角，状态指示器旁边

**完整版 → 简化版**：
1. 点击 **"🎯 简化版"** 按钮
2. 确认切换
3. ✅ 立即切换，无需重启

**简化版 → 完整版**：
1. 点击 **"🔧 完整版"** 按钮
2. 确认切换
3. ✅ 立即切换，查看套利功能

### 方法2：命令行切换 ⭐⭐

```bash
cd desktop-app/renderer

# 切换到简化版
mv index.html index_full.html
mv index_simple.html index.html

# 切换回完整版
mv index.html index_simple.html
mv index_full.html index.html
```

## ⚠️ 切换注意事项

### ✅ 保持不变
- 做市商运行状态
