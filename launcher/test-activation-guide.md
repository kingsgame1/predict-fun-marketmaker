## 🧪 功能测试清单

### 1. 输入框测试
- [ ] 点击输入框，可以输入文字
- [ ] 输入小写字母自动转为大写
- [ ] 自动添加连字符（每4个字符）
- [ ] 最多输入39个字符（8组×4字符+7个连字符）

### 2. 格式验证测试
**有效格式示例**（用于测试）:
```
ABCD-1234-EFGH-5678-IJKL-9012-MNOP-3456
```

**无效格式**:
- 少于39个字符
- 多于39个字符
- 格式错误（如：ABCD-1234）

### 3. 激活按钮测试
- [ ] 点击按钮后显示加载状态（旋转图标）
- [ ] 按钮在验证期间禁用
- [ ] 验证成功后显示绿色成功消息
- [ ] 验证失败后显示红色错误消息
- [ ] 成功后1.5秒自动跳转到主界面

### 4. 消息显示测试
**应该看到**:
- "正在验证激活码..." （加载中）
- "✅ 激活成功！有效期365天" （成功）
- "❌ 激活码格式无效" （格式错误）

## 🔍 如果窗口不可见

### 方法1: 使用 Mission Control
1. 按 **F3** 键打开 Mission Control
2. 查找 "Predict.fun" 窗口
3. 点击窗口激活

### 方法2: 检查 Dock 栏
1. 查看 Dock 栏是否有 Electron 图标
2. 右键点击 → 选择 "显示所有窗口"
3. 选择 "Predict.fun" 窗口

### 方法3: 强制激活窗口
```bash
./bring-to-front.sh
```

### 方法4: 重新启动
```bash
pkill -f "predict-fun-launcher"
npm start
```
